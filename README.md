## üìù Project Description

PARTYSERV is a web application built on the **ASP.NET framework**, designed to streamline and automate the operations of a private event and catering service provider.

The system aims to digitize and optimize the traditional event planning process, replacing manual record-keeping with an intelligent, data-driven platform. The application supports three primary user roles: general users, customers, and administrators.

* **Customers:** Can log in, book appointments, and receive confirmations.
* **Administrators (Admin):** Can manage event details, predict costs, allocate staff, track inventory, manage suppliers, and organize client meetings.

---

## ‚ú® Key Features

The system includes the following core functionalities:

* **User Management:** Registration, login, and appointment booking.
* **Event Planning:** Event cost estimation and resource planning.
* **Staff Management:** Staff allocation and coordination for emergencies.
* **Supply Chain Management:** Inventory and supplier management.
* **Analytics & Reporting:** Income forecasting and event calendar management.
* **Admin Dashboard:** Provides a comprehensive dashboard with statistics and detailed reports.
